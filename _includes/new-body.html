About

I'm a front end web developer working in London. Since 2017 I have been working for Experian where I build and optimize web pages, as well implement all of their web analytics tools, like Google Analytics.

I have always loved learning new things, and two years ago I began exploring different programming languages and teaching myself to code. I soon realised I really enjoyed it and needed to take it to the next level. This is when I began studying at General Assembley. I learned a huge amount there, and built a solid foundation on which to launch my career as a web developer.

I'm always improving as a developer, whether it's at Experian where I've learned a lot about web performance and A/B testing, or in my spare time doing courses on Python or React. I am always looking to learn new languages and frameworks, because more than just being a useful skill coding is a very rewarding skill. A goal of mine is to learn more about back end development, and become a full stack developer.
